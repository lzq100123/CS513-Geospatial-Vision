## How to run:
1. Put main.py and tilesystem.py in the same directory.
2. Make sure you have installed Pillow 5.0.0 or above
3. Open commander and type "python main.py" to run program with default setting 
or type "python main.py lat1 lon1 lat2 lon2" with two map coordinates (lat1, lon1) and (lat2, lon2)
